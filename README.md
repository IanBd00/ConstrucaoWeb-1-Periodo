# CSW-CC1N
Este repositório reúne materiais, exercícios e exemplos práticos elaborados ao longo da disciplina Construção de Software para Web, ministrada na Universidade Vila Velha (UVV). Os conteúdos apresentados refletem as atividades desenvolvidas em sala de aula e têm como foco a aplicação prática dos conceitos estudados.
O projeto tem como objetivo apoiar o processo de aprendizagem dos fundamentos do desenvolvimento web, por meio do uso das tecnologias HTML, CSS e JavaScript, amplamente empregadas na construção de aplicações web modernas.
